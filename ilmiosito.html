HTML e CSS

HTML (HyperText Markup Language) e CSS (Cascading Style Sheets) sono i fondamenti del web. HTML stabilisce la struttura e il contenuto di una pagina, mentre CSS gestisce lo stile e l'aspetto visivo, come colori, caratteri e disposizione degli elementi. Bootstrap

Bootstrap è un framework CSS gratuito e open-source, pensato per costruire siti web e applicazioni responsive in modo veloce. Offre una vasta gamma di componenti predefiniti, come pulsanti, modali, griglie e menu di navigazione, oltre a classi già pronte per personalizzare facilmente il design. JavaScript (JS) è un linguaggio di programmazione che rende le pagine web interattive e dinamiche. Consente di aggiungere funzionalità come aggiornamenti in tempo reale, animazioni, validazione dei moduli e interazioni con l'utente.
